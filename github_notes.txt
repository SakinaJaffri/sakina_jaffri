GIT HUB TERMINAL (notes)
1.	Cd sakina_projects (parent folder)
2.	Cd sakina_jaffri (repository local folder)
3.	Touch sakina110.txt (file created)
4.	Git init (to link local repository folder to github)
5.	Git status (to check the status of your file in repository folder)
6.	Git add ‘file name’ or ‘.’ (to add single file or entire folder)
7.	Git commit –m ‘your comments’ (to commit the file from repository)
8.	Git push origin main (to move the file to github)
